How does GSTMate Work?

 
GSTMate goes through a set number of processes to update your system
 
1.     Choose the Company to Run GSTMate on
2.     Backup your System (this may take some time depending on the size of your data)
3.     Set the GST Rate and determine of you want to increase the retail prices
4.     Update the System with the new settings
5.     Exit out of GSTMate
