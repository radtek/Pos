Frequently Asked Questions

 
Can the process be reversed?
 
Yes, if you need to change the GST back to 12.5% then rerun GSTMate and set the GST rate in Step 3 to 12.5 and recalculate the retail prices. Alternatively you will need to restore you backup from Step 2.
 
What does GSTMate Change?
 
GST will change the following setting and data
 
1.     All Stock Items will have the GST field changed to the new GST rate.
2.     All Menu Items will have the GST field changed to the new GST rate.
3.     If you have tick to recalculate the prices then all the selling prices in all menus will be increased to the new GST Rate. The system will remove the       old GST rate then add the new GST rate and recalculate the prices based on the rounding rules.
4.     Set the GST rate in the setup of the system to the new GST rate
 
Do I run GSTMate on all computer using menumate?
 
No, you only need to run GSTMate on the computer that you use Menumate Office on. If you only run Menumate Point of Sale on the computer then you DO NOT need to run GSTMate. If you have any questions email support@menumate.com or call 0800 657 300
 
 
When do I run GST Mate?
 
Run GSTMate at the end of the days trade on 30 September 2010 or before you start trading on the 1 October 2010
