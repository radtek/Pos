OnThemeChange Event
OnThemeChange is triggered when something related to theme support changes.

Applies to
TThemeServices

Declaration
TNotifyEvent = Procedure(Sender: TObject) of object;

Description
If the TThemeServices class is used in a standalone manner and got a window assigned (see also ClientWindow and Create) then OnThemeChange is triggered when the user or other applications change a setting related to themes (colors, styles etc.).
