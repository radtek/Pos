CM_DENYSUBCLASSING Global Constant
Component message to allow a control to exclude itself from subclassing.

Unit
ThemeMgr

Declaration
CM_DENYSUBCLASSING = CM_BASE + 2000;

Description
Some controls may not want to be subclassed. Be it because they have their own special painting or be it that subclassing would simply made trouble. A control can exclude itself from subclassing by catching the CM_DENYSUBCLASSING event and return a value <> 0 in the message's Result record member.
