OnControlMessage Event
Triggered whenever a message for a subclassed control comes in.

Applies to
TThemeManager

Declaration
TControlMessageEvent = Procedure(Sender: TThemeManager; Control: TControl; var Message: TMessage; var Handled: Boolean) of object;

Description
This event allows an application to modify a control's behavior just as would it has the control subclassed on its own. The event is triggered only for controls, which are subclassed by the Theme Manager and allows do additional processing (before the default processing takes place) or to fully take over message processing (in which case Handled should be set to True).
