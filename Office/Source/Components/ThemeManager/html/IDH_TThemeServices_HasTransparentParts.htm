HasTransparentParts method
Determines whether a certain element has transparent or alpha-blended parts.

Applies to
TThemeServices

Declaration
function HasTransparentParts(Details: TThemedElementDetails): Boolean;

Description
Many theme elements have at least non-rectangular bounds and some are even partially transparent. This requires to prepare the background accordingly before painting the element. Since this will consume more time than without the background painting HasTransparentParts can be used to determine if this special preparation is necessary at all.
