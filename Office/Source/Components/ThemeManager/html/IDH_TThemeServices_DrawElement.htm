DrawElement method
Draws a certain themed element.

Applies to
TThemeServices

Declaration
procedure DrawElement(DC: HDC; Details: TThemedElementDetails; const R: TRect; ClipRect: PRect = nil);

Description
This method is responsible to draw the element given by Details. DC is the target where on which to paint. Details describes what to draw and can be filled by GetElementDetails. R determines the area in the target where to draw the element. Note that the way how the element is drawn is determined by the theme. It can be stretched to fit, centered without stretching, or it can be tiled. The class has no control over the way, however. ClipRect is optional and limits painting to the given coordinates.
