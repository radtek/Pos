DrawParentBackground method
Used by partially transparent or alpha-blended child controls to draw the part of their parent that they appear in front of.

Applies to
TThemeServices

Declaration
procedure DrawParentBackground(Window: HWND; Target: HDC; Details: PThemedElementDetails; OnlyIfTransparent: Boolean; Bounds: PRect = nil);

Description
Window is the handle of the child control. Target is the place where to paint the parent's background to. Details is optional and only used if OnlyIfTransparent is true, which tells the method to check first if there are any transparent or alpha-blended parts in the theme element before causing the parent window to draw. Bounds is also optional and defines the area to be drawn (in child coordinates).
