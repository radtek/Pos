TThemeServices Object
Properties Methods Events
The TThemeServices class is a small wrapper around the themes APIs.

Unit
ThemeSrv

Declaration
TThemeServices = class(TObject)

Hierarchy
TObject <--TThemeServices

Subclasses
None

Description
Since themes are not available on all Windows platforms care must be taken to not break applications which are prepared for theme support on such platforms. This is one task of the TThemeServices class beside providing structured and convenient access to the theme APIs.

Introduced Public Properties
ClientWindow Contains the window handle given in the constructor.
Theme Returns the theme data handle for a certain theme class (e.g. window, toolbar, listview etc.).
ThemesAvailable Used to determine whether the class is running on a themed OS.
ThemesEnabled Used to determine whether theming is enabled for the application.

Introduced Methods
ColorToRGB Converts the given color to a theme color.
ContentRect Calculates the inner rectangle in a theme element, which is bound by the given bounding rectangle.
Create Constructor of the class.
Destroy Destructor of the class.
DoOnThemeChange Overridable method to trigger the OnThemeChange event.
DrawEdge Theme equivalent to the GDI DrawEdge API.
DrawElement Draws a certain themed element.
DrawIcon Draws an image within an imagelist based on a (possible) theme-defined effect.
DrawParentBackground Used by partially transparent or alpha-blended child controls to draw the part of their parent that they appear in front of.
DrawText Draws the text using the theme-specified color and font for the given element.
GetElementDetails
GetTheme Getter for the Theme property.
GetThemesEnabled Getter for the ThemesEnabled property.
HasTransparentParts Determines whether a certain element has transparent or alpha-blended parts.
PaintBorder Used to paint a theme border around a window.
UnloadThemeData Releases all theme data handles.
UpdateThemes Causes the class to release any loaded theme data and to ask again for the themes enable state.
WindowProc Internally used.

Introduced Public Events
OnThemeChange OnThemeChange is triggered when something related to theme support changes.
