DrawText method
Draws the text using the theme-specified color and font for the given element.

Applies to
TThemeServices

Declaration
procedure DrawText(DC: HDC; Details: TThemedElementDetails; const S: WideString; R: TRect; Flags, Flags2: Cardinal);

Description
DC is the target to draw into. Details defines the element to draw and can be filled with GetElementDetails. S contains the text to draw and R defines the text area. The parameter Flags is the same as "uFormat" in the GDI DrawText API. Flags2 defines additional drawing options. Currently only DTT_GRAYED is defined.
