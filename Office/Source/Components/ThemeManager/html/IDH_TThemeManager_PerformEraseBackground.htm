PerformEraseBackground method
Special method to make a control's background transparent.

Applies to
TThemeManager

Declaration
procedure PerformEraseBackground(Control: TControl; DC: HDC);

Description
This method can be used to paint a control's background for controls, which do not have an associated window or whose parent cannot handle the WM_PRINT message. It is an alternative to the TThemeServices.DrawParentBackground method, but should be used only in exceptional cases.
