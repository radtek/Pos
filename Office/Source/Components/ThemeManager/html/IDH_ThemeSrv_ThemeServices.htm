ThemeServices Function
Returns a global instance of a TThemeServices class.

Unit
ThemeSrv

Declaration
Function ThemeServices: TThemeServices;

Description
For simplification it is just easier to have one global Theme Services instance instead creating one for each control. There is only one exception, namely when a window needs own theme data, which differs from the global settings. Just like with Printer the ThemeServices function creates an instance only on demand and can so automatically be removed by the linker if the theme services are never used.
