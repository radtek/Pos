ThemesEnabled property
Used to determine whether theming is enabled for the application.

Applies to
TThemeServices

Declaration
Property ThemesEnabled : Boolean;

Description
The result is influenced by several factors. Firstly, the application runs on a theme aware OS. Secondly, a theme is active in the system. Thirdly, the application has an associated manifest (must be imported by the application or given as external manifest) and fourthly, theming is not disabled for the user and window. Note also that there are options to enable or disable certain theme operations. These options can be retrieved by GetThemeAppProperties.
